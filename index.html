📚 How to Install

Get set up in a few minutes

Step 1 — Create an account

On the login screen, click Sign up and register with your Taft email address and a password of your choice.

Step 2 — Import from Canvas

After signing in, open the Canvas Calendar, then click Calendar Feed at the bottom-left of the page. Copy that URL and paste it into the setup wizard.

Step 3 — Install the Chrome Extension (optional)

The extension lets you import assignments directly from Canvas course module pages with one click. Install it from the Chrome Web Store, then navigate to any Canvas course page and click the extension icon to sync assignments.
