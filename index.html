How to Install

Get set up in a few minutes

Step 1 — Create an account

On the login screen, click Sign up and register with your Taft email address and a password of your choice.

Step 2 — Install the Chrome Extension

Download the TaftTasks extension and add it to Chrome. This is how you import courses and assignments from Canvas.

Step 3 — Import your courses

Open any page on Canvas, click the TaftTasks extension icon, sign in, and hit Find Courses. Select the courses you want and click Scrape Selected to import all assignments.

Step 4 — Stay up to date

Use Refresh All Courses in the extension anytime to pull in new assignments. You can also add assignments manually from the dashboard.
