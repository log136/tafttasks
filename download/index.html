TaftTasks Extension

A Chrome extension that imports your Canvas assignments directly into TaftTasks.

⬇ Download Extension

Installation

1
Download and unzip the file above.
2
Open Chrome and go to chrome://extensions.
3
Enable Developer mode using the toggle in the top-right corner.
4
Click Load unpacked and select the unzipped extension folder.
5
Navigate to a Canvas course page and click the Taft icon in your toolbar to import assignments.
